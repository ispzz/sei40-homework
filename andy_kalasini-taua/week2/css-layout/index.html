<span> tags © are generic container tags which are "inline" by default. See what I mean?

static (default):
the position is whatever the browser calculates for it in the page flow; you can't change it later with top/left CSS offsets
relative:
Offsets do work, and are calculated relative to the default (static) position in the page flow
(parent)
absolute:
Offsets work, and are calculated relative to the nearest containing element (ancestor) which has a specific non-static position value set.... otherwise relative to the "body" (whole document)
fixed:
Offsets are calculated relative to the entire browser window (viewport) - i.e. the element does NOT move when you scroll the document
